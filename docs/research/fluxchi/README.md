# FluxChi Research

This directory is the repo-side entrypoint for FluxChi research work.

Detailed paper notes live in Zotero, not here.

Use this folder to keep three things stable:

- what FluxChi is currently researching
- how Zotero is organized for this project
- how research connects to product and evidence loops

## Current focus

Priority order:

1. `Multimodal State Detection`
2. `Intervention Strategy`
3. `Review & Evidence Design`

## Source of truth

- Paper metadata, PDFs, annotations, and structured notes: Zotero library `FluxChi`
- Project-facing structure and rules: this folder

## Files

- `zotero-schema.md`
  - Collections, tags, note template, and operating rules
- `research-questions.md`
  - Current questions that decide what papers are worth collecting

## Operating rule

Do not dump long literature summaries into repo markdown.

Every useful paper should end up producing one or more of:

- a `FluxChi Implication`
- a `Testable Hypothesis`
- a product rule candidate
- an evidence capture idea

The intended loop is:

`paper -> implication -> hypothesis -> product rule -> evidence -> next paper`
