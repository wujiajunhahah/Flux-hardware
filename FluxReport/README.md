# Project Flux Delivery Package

This folder gathers the key information about the Waveletech EMG wristband integration completed in this repository, including hardware description, software architecture, usage instructions, and our collaboration dialogue.

## Contents

- `introduction.md` – Overview of Project Flux, hardware notes, and the final feature set.
- `instructions.md` – Step-by-step guide for recording, training, inference, and LibEMG streaming.
- `conversation_log.md` – English digest of our collaboration so far.
- `中期检查表.md` – 研究生中期检查表（深圳技术大学 未来技术学院 工业设计）。
- `文献DOI交叉验证核查表.md` – 文献 DOI 交叉验证、数据库核查与原文访问指引。

All files are written in English for dissemination or archival purposes. Source code in the repository remains unchanged.
